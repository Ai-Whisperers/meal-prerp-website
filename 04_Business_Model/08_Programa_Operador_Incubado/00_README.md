## Qué es este modelo

**Un operador independiente** (no un empleado nuestro) corre un servicio
**tipo HelloFresh local, congelado**, y **nosotros somos uno de sus clientes**.

El repo completo actúa como **playbook transferido al operador** — compras
mayoristas, desposte, crash cooling, manual del chef, directorio de
proveedores, guías de empaque, etc. Nosotros aportamos el playbook +
cartera semilla + mentoría; el operador aporta tiempo, cocina, desposte,
marca propia.

## Por qué este modelo supera al anterior

| Dimensión | Modelo v1 (Chef Empleado) | Modelo v2 (Operador Incubado) |
|---|---|---|
| Gestión laboral (IPS, aguinaldo, riesgo) | Nuestra | Del operador |
| Costo mensual para Iván | ~Gs. 4,3 M/mes (chef formal) | ~Gs. 650 k/mes (6 porciones/sem × Gs. 24k) |
| Clientes del servicio | 5 hogares fijos (co-op cerrado) | Cartera abierta del operador |
| Rol de Iván | Operador + cliente ancla | **Cliente + mentor ligero** |
| Incentivo del cocinero | Sueldo fijo + bonus | Dueño del upside |
| Exit / cambio | Contratar / despedir | Cambiar de proveedor |

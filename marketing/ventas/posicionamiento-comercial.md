## Categoría

**Recurring freezer stock** + optional mise — NOT restaurant, NOT meal kit (client cooks), NOT personal chef.

| | HelloFresh (kit) | Industrial freezer | PedidosYa | **Nosotros** |
|--|------------------|--------------------|-----------|--------------|
| Cliente cocina | Sí (~40 min) | No | No | No |
| Precio/plato PY | — | Gs. 20–30k | Gs. 55–85k | Gs. 25–28k |
| Calidad | Variable | Industrial | Variable | Chef batch IGA |
| Stock semana | No | Sí | No | Sí |

**Sweet spot:** Gs. 28.000 público — arriba industrial, debajo delivery, fracción del chef a domicilio.

## Promesas públicas (freezer)

1. Duración 7d heladera / 30d freezer — reposición si falla producción  
2. Etiquetado completo + instrucciones regeneración  
3. Sin conservantes químicos — crash cool + vacío  
4. Feedback → plato sale del menú
